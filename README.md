
A Comparative Study of distinction between Hybrid Kmeans Decision Tree and Standard Decision Tree on FPGA Hardware Accelerator

Trained a Hybrid Kmeans Decision Tree on Google Colab in python language and compared it to standard Decision Tree present on the libraries of Colab. Next, using this trained model, converted it into C++ to see different resource usage and latency and executed on Vitis HLS. Then exported RTL as IP to see the block design of the model on Vivado and checked synthesizability.
